Aqui começamos nossos estudos de git e github
Principais comandos do git:
    - git status - Verifica estado dos arquivos do nosso projetos
    - git add - prepara o aruivo para ser commitado
    - git commit -m 'mensagem' - salva o arquivo na branch atual salvando alterações
    - git branch - visualiza as branchs existentes
    - git branch <nome_da_nova_branch> - insere uma nome linha de progresso paralela à linha principal ou master
    - git checkout <nome_da_branch> - alterna entre as branchs existentes
    - git merge <nome_da_branch> - insere as alterações feitas na branch secundária para a branch principal
    - git checkout -b <nome_da_nova_branch> - cria uma nova branch e entra direto na branch
    - git log - mostra os commits feitos no projetos
    - git log --graph - mostra uma espécie de log grafico com todos os commits
    - git log -- graph --oneline - mostra o log grafico simplificado
